# Data-Engineer-Weather
Se consume de la API REST de https://openweathermap.org para consultar los datos
metereologicos de una ciudad dada, en el sistema metrico pedido (metrico o imperial).
# Utilizacion
´´´Python3 script.py [city_id] [units]´´´

Para consultar los id de ciudad, descargar el Json de la pagina de openweather.org (https://bulk.openweathermap.org/sample/city.list.json.gz)
